Our principles

Privacy-first

Most tools run entirely in your browser. We don’t ask for sign-ups or personal details. Where an API is needed, we keep it transparent.

Performance

Minimal JavaScript, optimized media, and accessible color contrast. Pages are designed for fast FCP/LCP and low CLS.

Clarity

Each tool explains inputs, outputs, and assumptions-with examples where helpful. No dark patterns, no clutter.

Accessibility

Keyboard focus styles, readable fonts, and high contrast UI. We aim for WCAG AA where practical.

Made for India

India-first finance: Income Tax, GST, EMI/SIP, IFSC/PAN utilities-plus global formatting and developer tools.

Open to feedback

Got an idea? Suggest a tool or report a bug. We iterate quickly.

FAQ

Do you store or send my data to a server?

No. Most tools run entirely in your browser. For tools that need network calls (e.g., IFSC lookup), we indicate clearly. We don’t ask for logins or personal details.

Are your finance tools India-focused?

Yes. We prioritize India-first calculators like Income Tax, GST, EMI, SIP, and IFSC, while also offering global utilities such as JSON Formatter, Base64 Encoder, and Currency Converter.

Can I suggest a tool or report a bug?

Absolutely-use the Contact page to share ideas, report issues, or request new features.
